# Highlights of the release

Scala 3.8.1 is a hotfix release for [3.8.0](./3.8.0.md).
It addresses runtime regressions found after the 3.8.0 artifacts were release, specifically JVM linkage errors affecting a small subset of Scala 2 specialized types and members.

## Backported fixes

- Revert addition of new base trait to ReturnThrowable [#24975](https://github.com/scala/scala3/pull/24975)
- Use uninlined version of Scala 2 standard-library in ScalaLibraryPlugin [#25008](https://github.com/scala/scala3/pull/25008)
- Add missing specialized method in NumericRange [#25020](https://github.com/scala/scala3/pull/25020)
- Correct the list of files to copy from Scala 2 [#25033](https://github.com/scala/scala3/pull/25033)
